QR payments via PayPal

== Description ==
Accept payments with a QR code. Payment goes through PayPal payment gateway. 
You need to install the Woocommerce plugin.

Use the correct phone number and address even for testing!

== Installation ==

1. Upload `qr-payments-via-paypal.zip` to the `/wp-content/plugins/` directory
2. Unpack the archive
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Enable the payment method on the page WooCommerce > Settings > Payments
5. Click on the 'Set Up'
6. Fill in the plugin data
7. Use
